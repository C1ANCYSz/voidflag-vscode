# Changelog

## [0.0.3] - 2026-03-19

### Added

- JSON schema validation for `voidflag.config.json` config files with autocomplete and descriptions for all fields:
  - `voidflagDir` — directory where VoidFlag files are stored
  - `lang` — output language
  - `schemaPath` — path to the `.vf` schema file
  - `defaultEnv` — default environment name
  - `devPort` — development server port
  - `environments` — environment name to env variable key mapping
- Syntax highlighting for `.vf` files
